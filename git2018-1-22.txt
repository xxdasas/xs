1  初始化 git init (此文件不要动) 
2 git status(查看文件状态)
3 git add . (当前文件夹所有的文件提交) git add 文件名
4 删除文件 git rm 文件名 (删除必须是commit 之后的文件) 删除完事要commit
5 查看当前用户名 git config user.name  查看当前用户 email  git config user.email
6 清除 origin            git remote-v     git remote remove origin
7  配置origin  git remote add origin 地址
